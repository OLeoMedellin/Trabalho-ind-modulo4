3 perguntas são propostas sobre o banco de dados sendo estas e suas respectivas respostas:

Existem outras entidades além destas 3?
Sim, pode se acrescentar entidades como professores, locais ou formatos do curso, matrícula, etc.

Quais são os principais campos e tipos?
Os principais campos são id, nome, data de nascimento e turmas. Os dois tipos são chaves primária e secundária

Como estas entidades estão relacionadas?
Através de chaves estrangeiras (FK)
